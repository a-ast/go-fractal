## Web server app

Run

```
cd cmd/server
go build
./server
```


### Url parameters

* t: fractal type. Options: julia, mandelbrot, burningship
* p: predefined colour palette. Options: arcticsun, electro
* w: image width
* h: image height
* z: zoom, default: 1.0
* cx: center by X 
* cy: center by Y
* re:
* im:
* er: escape radius
* mi: max iterations
